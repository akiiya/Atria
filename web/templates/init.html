📖 如何获取 API Key
  1. 前往 my.telegram.org
  2. 使用你的 Telegram 手机号登录
  3. 进入 API development tools
  4. 创建 application，复制 api_id 和 api_hash
💡 提示：通常只需申请一套 API Key。之后接入其它 Telegram 账号时，直接使用手机号 + 验证码登录，无需重复申请。
🌐 网络建议：申请时建议使用干净、稳定、低风险的网络环境。网络出口地区建议与手机号常用地区保持一致，以提高成功率。

安全说明

🔑
本地密钥

Atria 使用本地 master key（data/secret.key）加密敏感数据。请务必备份此文件，丢失后无法恢复已保存的 API Key 和 Session。

🔐
加密存储

api_hash、Session 等敏感数据使用 AES-256-GCM 加密保存，数据库中不存储明文。

🚫
无默认密码

Atria 不提供任何默认管理员账号，必须由用户首次访问时自行设置。
