# 人工浏览器验收清单

本文档列出自动化测试无法覆盖、需要真实浏览器验证的关键路径。

每次发布 main 前必须逐项验收。

**当前基线：** commit `209acc4`，migration v13，i18n 341 key/locale × 10 locale

## 前置条件

1. 启动 `bin/atria.exe serve`
2. 浏览器打开 `http://localhost:8080/app/`
3. 已登录管理员
4. 已接入至少一个 Telegram 账号
5. Runtime 状态为 live
