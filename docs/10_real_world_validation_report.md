## 发现的问题

### 问题 1：SecretKeyFile 不随 DataDir 变化

- **编号：** BUG-001
- **现象：** 设置 ATRIA_DATA_DIR 环境变量后，secret.key 仍创建在默认 ./data/ 目录
- **影响：** 使用自定义数据目录时密钥文件位置不正确
- **修复状态：** ✅ 已修复
- **修复方式：** 当 ATRIA_SECRET_KEY_FILE 未设置且 DataDir 变化时，自动更新 SecretKeyFile 路径

### 问题 2：Windows 下 secret.key 权限警告

- **编号：** ISSUE-001
- **现象：** smoke test 报告 secret.key 权限为 644，非 600
- **影响：** Windows 不支持 Unix 文件权限，这是预期行为
- **修复状态：** 不需修复（平台限制）
- **说明：** Linux/macOS 环境下权限应为 0600
